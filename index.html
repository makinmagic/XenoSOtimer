🕓 What is XenoSO Timer?

The XenoSO Timer is a lightweight desktop companion for XenoSO players who want precise tracking and alerts for SMOs, job shifts, egg hours, and seasonal events - all in one small app!

Whether you’re working on jaaaaaaams, managing your Diner shifts, or waiting for Egg Hour, this timer helps you stay perfectly synced with in-game time.

⚙️ Key Features

  • 🏭 Multi-Timer Interface - Manage multiple activities from one window.
  • 🕕 Egg Hour Countdown - Automatically tracks the next egg hour.
  • 🔝 Stays on Top - Keep sight of your timer no matter what you're doing.
  • 🔔 Audio Alerts - Choose whether you want to receive audio alerts or not.
  • 🚀 Lightweight & Offline - Runs standalone, no installation required.

💡 How It Works

XenoSO Timer synchronizes with the in-game clock and runs locally. Each timer is independent - start the one for your SMO or candy bowl, and it will alert you when you can collect your monies or your candies.

  • Factory / Diner / Club: Track shift start times automatically. No job abandonment! 💼
  • Egg Hour: Never miss your eggs again! 🥚
  • Candy Timer: Keeps you from getting zapped! ⚡

💾 Download & Installation

  1. Click on the Download link.
  2. Extract the files and run XenoSOtimer.exe. I recommend adding a Desktop shortcut.
  3. That's it! No installation needed - just launch and play.

⚠️ Windows may warn you about "unverified software." Click “More Info → Run Anyway” to proceed safely.

❓ FAQ

Is it safe to use?

Yes. XenoSO Timer is open-source and hosted on GitHub. You can inspect the code before downloading or using it.

How do I install updates?

All you need to do is delete the old files, unzip the new ones, and use the new XenoSOtimer.exe! Simple as that 🙌

What do I need to run it?

XenoSO Timer requires .NET 8.0, but since this is included with your XenoSO install, if you can play XenoSO, you can run the timer.
