🕓 What is XenoSO Timer?

The XenoSO Timer is a lightweight desktop companion for XenoSO players who want precise tracking and alerts for SMOs, job shifts, egg hour, AFK timeouts, and in-game events - all in one small app!

Whether you’re working on jaaaaaaams, managing your Diner shifts, or waiting for Egg Hour, this timer helps you stay perfectly synced with in-game time while looking like part of the game itself.

⚙️ Key Features

  • 🏭 Multi-Timer Interface – Manage multiple activities (SMOs, jobs, and more) from one window.
  • 🕕 Egg Hour, Jobs, and Events Countdowns – Never miss an egg, shift, or bingo again!
  • 💤 AFK Timer Interruptor™ – Warns you before you’re logged out for inactivity.
  • 🔝 Always on Top – Keep your timer visible during gameplay.
  • 🚀 Lightweight & Easy to Use – Runs standalone, no installation required.

💡 How It Works

XenoSO Timer synchronizes with the in-game clock and runs locally. Each timer is independent - start one for your SMO, Egg Hour, or an event, and it will alert you when it’s time.

  • Factory / Diner / Club: Track shift start times automatically. No job abandonment! 💼
  • Egg Hour: Don't miss a single minute of chicken productivity! 🥚
  • Events: Countdown to events straight from the XenoSO dashboard! 🎉
  • AFK Timer Interruptor™: Avoid the AFK monster with multiple sound alerts! ⚡

💾 Download & Installation

  1. Click on the Download link.
  2. Extract the files and run XenoSOtimer.exe. I recommend adding a Desktop shortcut.
  3. That's it! No installation needed - just launch and play.

⚠️ Windows may warn you about "unverified software." Click “More Info → Run Anyway” to proceed safely.

❓ FAQ

Is it safe to use?

Yes. XenoSO Timer is open-source and hosted on GitHub. You can inspect the code before downloading or using it.

How do I install updates?

Simply delete the old files, unzip the new ones, and use the new XenoSOtimer.exe! Simple as that 🙌

What do I need to run it?

XenoSO Timer requires .NET 8.0, but since this is included with your XenoSO install, if you can play XenoSO, you can run the timer.
